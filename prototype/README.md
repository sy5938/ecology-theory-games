# 光之林 MVP 原型

这是一个无依赖的浏览器原型，用来验证“光照—生活史—竞争—共存”的核心循环。

## 运行

在仓库根目录执行：

```bash
python3 -m http.server 8000
```

然后打开 <http://localhost:8000/prototype/>。

## 当前范围

- 12 个真实物种，3 类抽象策略；
- 单局 6 个物种：玩家 1 个、系统 5 个；
- 60×40 细网格；
- 林下层动态光照；
- 种子繁殖、生长投资、维持策略；
- 生活史阶段、连续树高/胸径增长；
- 自疏限制；
- 局部台风预警与林冠死亡；
- 群落指标和结算反馈。

## 本轮试玩反馈修正

- 放慢种子、幼苗、幼树之间的阶段转移；
- 采用 650ms 回合推进动画；
- 增加生活史阶段进度条；
- 增加“生态过程”日志，显示指令、个体变化和台风；
- 增加明确的结算弹窗和再来一局入口。
