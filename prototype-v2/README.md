# 像森林一样思考 · V2 可玩原型

> THROWAWAY PROTOTYPE：验证“单物种生命史投资组合 + 连续森林演化”是否具有可玩性。确认方向后应吸收有效结论并删除失败布局。

## 启动

在仓库根目录运行：

```bash
npm install
npm run demo
```

打开终端显示的本地地址。开发模式下底部切换器提供三个布局，也可直接访问：

- `?variant=A`：投资工作台；
- `?variant=B`：森林优先；
- `?variant=C`：研究仪表盘。

## 本轮验证问题

1. 持续配置“生长 / 繁殖 / 储备”是否比三个回合按钮更有决策感？
2. 玩家能否从动态光照、个体状态和种群曲线理解投资回报？
3. 单物种控制、其他物种自动运行是否形成足够的不确定性？

## 已实现边界

- 密闭林冠初始条件；
- 单局 6 个物种，玩家控制 1 个；
- 连续生态时间步，暂停与倍速；
- 动态碳收入、基础维持、生长/繁殖/储备配置；
- 个体 hover/click 和空地点选；
- 林冠遮阴、死亡后林窗变亮；
- 台风模糊预警，不提供专用事件技能；
- 种群时间序列与树高分布；
- 三分钟结算检查点，之后可继续。
