# 像森林一样思考 · V3 可玩原型

单人森林演替游戏：玩家只经营一个真实物种，通过持续配置生长、繁殖和储备，在光照、竞争、病原菌、扰动与优势种虫害之间做生命史投资权衡。多物种共存与单种优势都是可能结局。

## 启动与关闭

在仓库根目录运行：

```bash
npm install
npm run demo
```

打开终端显示的本地地址。关闭时回到运行命令的终端，按 `Ctrl+C`。

## V3 可玩内容

- 三种开局：密闭森林、稀疏森林、先锋定殖；
- 密闭森林约 360 个体，实际林冠投影覆盖率验收为 ≥95%；
- 所有个体均为圆形；玩家物种由更大的白色圆环包围，大小按树高对数映射；
- 48 m × 32 m 米制标尺、滚轮缩放、平移、复位、林冠/林下过滤；
- 个体点击、多选、玩家幼苗/幼树一次性移栽，以及可筛选排序的属性表；
- 光照、竞争、同种病原菌、健康、碳储备与死亡原因分开计算；
- 局部台风、全图暴雨、优势种专性虫害和死亡后 5 秒红叉；
- 主动能力“诱导防御”和“集中结实”；
- 第 30 年唯一一次强制结算；继续后解锁 8×/16×，随时可主动生成“第 N 年森林演替结算”；
- ZIP 导出个体快照、物种年度历史、事件、报告与元数据。

## 验证命令

```bash
npm run test:model
npm run test:browser
npm run build:demo
```

模型测试使用固定随机种子检查开局覆盖率、健康目标、虫害、结算和 ZIP；浏览器测试使用本机 Chrome 检查地图、属性定位、分层、缩放与报告交互。
