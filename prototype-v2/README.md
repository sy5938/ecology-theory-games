# 像森林一样思考 · V2 可玩原型

> THROWAWAY PROTOTYPE：验证“单物种生命史投资组合 + 连续森林演化”是否具有可玩性。确认方向后应吸收有效结论并删除失败布局。

## 启动

在仓库根目录运行：

```bash
npm install
npm run demo
```

打开终端显示的本地地址。当前版本已将三个试验布局收敛为一个统一仪表盘，不再提供容易误触的布局切换器。

## 本轮验证问题

1. 持续配置“生长 / 繁殖 / 储备”是否比三个回合按钮更有决策感？
2. 玩家能否从动态光照、个体状态和种群曲线理解投资回报？
3. 单物种控制、其他物种自动运行是否形成足够的不确定性？

## 已实现边界

- 密闭林冠初始条件，约 252 个混合生活史阶段个体；
- 单局 6 个物种，玩家控制 1 个；
- 连续生态时间步，暂停与倍速；
- 动态碳收入、基础维持、生长/繁殖/储备配置；
- 投资控制位于地图上方，提供“抢占林冠 / 扩散下一代 / 稳健生存”快速策略和本期实际投入；
- 画布级 hover/click 和空地点选；Shift / ⌘ 可多选一组个体并查看平均轨迹；
- 玩家幼苗/幼树可拖动移栽一次，并承担 4% 健康代价；
- 高度达到 10 m 即进入冠层；遮阴强度继续随树高增加，死亡后林窗平滑变亮；
- 光照适宜度直接影响个体健康；同种聚集产生病原菌压力，降低幼苗建立并损害健康；
- 局部台风与全图暴雨提前预警，储备可缓冲扰动风险；
- 种群时间序列与树高分布；
- 顶栏可随时打开阶段报告；三分钟自动结算，展示种群、健康、冠层、光照、病原菌和事件过程，之后可继续。

## 当前界面结构

- 左侧：动态林下光照地图与环境事件；
- 右侧：玩家仓位、点位检查与动态图表；
- 上方：横向生命史投资组合和快速策略，结果在下方地图中持续显现。
